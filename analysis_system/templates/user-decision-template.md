## 📋 使用说明

### 🎯 模板用途
- 在每轮分析循环的第9步使用
- 帮助用户评估分析进展并做出决策
- 确保分析过程的可控性和用户参与度

### 📝 填写指导
1. **成果描述**: 简洁列出本轮的主要分析成果
2. **目标达成度**: 评估相对于总体计划的完成百分比
3. **关键发现**: 总结本轮最重要的发现
4. **问题分级**: 将遗留问题按优先级分类
5. **决策选择**: 用户根据情况选择后续行动

### 🔄 决策结果处理
- **选择1**: 创建下一轮分析环境，制定下轮分析计划
- **选择2**: 调整当前轮次分析范围，重新执行部分分析步骤
- **选择3**: 进入阶段三汇总报告流程
- **选择4**: 暂停当前分析，收集额外信息后决定是否继续
